# Deep Learning 项目仓库说明

本仓库用于整理和管理本人在 深度学习、计算机视觉及算法学习 过程中产生的代码与实验内容，涵盖基础学习、课程作业以及科研相关实现。

## 一、目录结构说明
### 0. coding

该目录用于存放今后进行深度学习任务的核心代码，包括但不限于 *YOLO*、*PyTorch*等相关实现。
在可预见的时间内，这是使用频率最高的目录。
所有未来与科研相关的代码将统一在此目录中开发、维护与发布。

### 2. python_basis_excrise

该目录为研 *0* 阶段暑假期间自学 *Python* 基础 所建立的文件夹。
主要内容包括：

*Python* 基础语法与编程练习

后续用于非深度学习算法训练的代码实现

### 3. python_cv2_basis

该目录主要基于 *OpenCV（cv2）* 这一传统图像处理库进行实验与学习。
内容包括：

研一期间*《图像处理算法》*课程的全部作业

自学过程中整理的 *cv2* 相关实验与代码示例

### 4. Deeplearning_basis

该目录用于保存自学深度学习过程中的相关代码与学习资料，目前仍在持续更新中。
当前学习重点主要集中在 *CS231n*：卷积神经网络与视觉识别 课程内容。

### 5. homework_BP

该目录主要用于提交嵌入式系统课程作业，实际使用价值较低。
内容主要包括 多层感知机*（MLP）* 等基础神经网络的实现代码。

### 6. class_ZhuQiBing

该目录包含研一期间学习 深度神经网络 课程时，由任课教师布置的课程作业及相关代码。

# Deep Learning Project Repository

This repository organizes codes and materials related to deep learning, computer vision, and algorithm practice throughout the research and study period.

## First.Directory Structure
### 0. coding

This directory contains core code for deep learning tasks, including but not limited to YOLO and PyTorch implementations.
It is expected to be the most frequently used directory over the next several years.
All future research-related code will be developed, maintained, and released here.

### 2. python_basis_excrise

This directory was created during the pre-master (Year 0) summer for self-study of Python fundamentals.
It includes exercises covering Python syntax and basic programming concepts.
In the future, it will also store implementations of non-deep-learning algorithms and general code training tasks.

### 3. python_cv2_basis

This directory focuses on traditional image processing using OpenCV (cv2).
It contains:

All assignments from an Image Processing Algorithms course taken in the first year of graduate study

Additional OpenCV-related experiments and self-learning materials

### 4. Deeplearning_basis

This directory stores materials and code from the process of self-learning deep learning and is continuously updated.
Current learning efforts are mainly based on the CS231n: Convolutional Neural Networks for Visual Recognition course.

### 5. homework_BP

This directory was mainly used for submitting assignments for an Embedded Systems course.
It includes basic implementations of a Multilayer Perceptron (MLP) and related experimental code.

### 6. class_ZhuQiBing

This directory contains assignments and related code from a Deep Neural Networks course taken during the first year of graduate study, provided by the course instructor.
